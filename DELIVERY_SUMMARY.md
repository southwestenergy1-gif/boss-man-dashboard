## Getting Started

### 1. Installation (2 minutes)
```bash
cd boss-man-dashboard
npm install
```

### 2. Development (30 seconds)
```bash
npm start
```
Opens browser at http://localhost:5173

### 3. Explore (5 minutes)
- Click business tabs
- Hover over cards
- Test on mobile (F12 → device toggle)
- Watch animations

### 4. Customize (Optional - 5 minutes)
Edit `src/data/mockData.js` and change any numbers:
```javascript
newThisWeek: 15      // ← Change this
revenueTarget: 50000 // ← Or this
```
Save file → Dashboard updates instantly

### 5. Deploy (Optional - 30 minutes)
See DEPLOYMENT_GUIDE.md for:
- Netlify (easiest - 5 min)
- Vercel (fast - 10 min)
- AWS S3 (scalable - 30 min)
- Your own server (flexible)
